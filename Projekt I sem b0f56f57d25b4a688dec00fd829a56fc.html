Imię i nazwisko studenta

Mateusz Urbanek

Temat zadania

Dany jest plik binarny zawierający dane pracowników: imię, nazwisko, adres, stanowisko, staż pracy, pobory zasadnicze, dodatki. Napisz program, który umożliwi obliczenie wysokości wypłaty, obliczy i wypisze średnią miesięczna wypłatę dla podanego stanowiska, dodanie danych nowego pracownika, usunięcie danych, edycję a także posortowanie elementów pliku rosnąco lub malejąco (wybór użytkownika) wg. podanego kryterium. Podaj rozwiązanie bez tablic pomocniczych.

1. Specyfikacja wymagań

System służy do wspomagania zarządzania kadrami poprzez naliczanie wynagrodzeń oraz przechowywanie wrażliwych danych o pracownikach w firmach do 60 osób.

System tworzy baze danych w pliku binarnym (o rozszerzeniu. dat) która przechowuje dane pracowników w rekordach o poniższym schemacie :

ID – numer pracownika w kolejności od początku do końca pliku. Pesel – składa się z 11 cyfr.

Imię – maksymalnie 30 znaków. Nazwisko – maksymalnie 40 znaków. Adres – maksymalnie 50 znaków. Stanowisko – maksymalnie 40 znaków. Staż pracy – maksymalnie 6 cyfr. Pobory zasadnicze – maksymalnie 9 cyfr. Dodatki – maksymalnie 9 cyfr.

Gdy użytkownik wprowadzi liczbę 8 cyfrowa (na przykład 12345678), system odczyta ten pesel jako 00012345678.
Najmniejsza jednostka pola pobory oraz dodatki to jeden grosz. Oznacza to, że na przykład dodatki = 1000 będą oznaczać 1000gr., czyli 10zł . Z tego względu maksymalna wartość tych pól musi być mniejsza, niż 10 mln zł.

2. Opis funkcji

void dodaj_pracownika(string nazwa);

Wywołuje funkcje wczytaj_pracownika. Przekazuje jej wartość wraz z nazwa do funkcji przekaz_pracownika.

struct pracownik wczytaj_pracownika(string nazwa);

Funkcja oczekuje od użytkownika wpisania każdego elementu struktury pracownik po czym ja zwraca.

void przekaz_pracownika(string nazwa,pracownik p);

Zapisuje strukturę (pracownik p) na końcu pliku binarnego (string nazwa).

void usun_pracownika(string nazwa,int id);

1. Tworzy nowy plik z dopiskiem "new_" do nazwy.

2. Zapisuje do tego pliku wszystkie struktury z wyjątkiem tego którego kolejność jest równa argumentowi id.

3. Usuwa stary plik (bez "new_") po czym zmienia nazwę nowego na stary.

void wypisz_pracownikow(string nazwa)

Wypisuje wszystkie struktury pracownik z pliku (string nazwa);

void wypisz_pracownikow_wizualnie(string nazwa);

Wypisuje strukturę pracownik z pliku (string nazwa) w atrakcyjnej wizualnie formie.

void wypisz_pracownikow_naglowek();

Wypisuje nagłówek w kolorze potrzebny do wyświetlenia funkcji wypisz_pracownikow_wizualnie.();

void edycja_pracownika(string nazwa,int n);

Wywołuje funkcje wczytaj_pracownika(nazwa) po czym zapisuje ja na indeksie podanym przez użytkownika.

void zamien_pracownika(string nazwa,int a,int b);

Zamienia 2 struktury pracownik z pliku (string nazwa) o indeksach a oraz b.

void bubbleSort(string napis,int n,int m)

Sortuje plik algorytmem bąbelkowym gdzie m jest sposobem sortowania a n elementem.

int ilosc_pracownikow(string napis);

Zwraca ile struktur (pracownik) znajduje sie w pliku.

int sprawdz_pesel(string nazwa,long long int pesel);

Sprawdza czy podany pesel jest juz w pliku (zwraca 0 w przypadku fałszu, 1 w przypadku prawdy).

void oblicz_wyplate(string nazwa,int n);

Liczy i wypisuje średnia miesięczną wypłatę dla podanego pracownika (pobory zasadnicze + dodatki).

string losowe_imie();

Losuje i zwraca 1 z 15 imion.

string losowe_nazwisko();

Losuje i zwraca 1 z 15 nazwisk.

string losowe_adres();

Losuje i zwraca 1 z 15 adresów.

string losowe_stanowisko();

Losuje i zwraca 1 z 15 stanowisk

struct pracownik generuj_pracownika(int maxi_i,double maxi_d);

Generuje strukturę pracownik składająca się z predefiniowanych danych znakowych oraz losowych liczb.

max_i to maksymalna wartość zmiennej typu int dla student.staz_pracy.

maxi_d to maksymalna wartość dla student.pobory_zasadnicze i student.dodatki.

void dodaj_gen_pracownika(string nazwa);

Wywołuje funkcje generuj_pracownika i zapisuje ja do pliku.

long long int potega(int podstawa,int wykladnik);

Potęguje podstawę do wykładnika (dla liczb całkowitych).

double rand_d(double fMin, double fMax);

Losuje zmienna typu double od fMIn do fMax.

long long int n_number(int n,string nazwa);

Zwraca n cyfrowa liczbe.

int dodatnia(long long int x);

Sprawdza czy liczba jest nieujemna.

bool kontrola_poprawnosci(long long int x);

Kontrola poprawności dla liczb całkowitych.

int wybierz_int();

Wczytuje i sprawdza poprawność zmiennej typu int.

void start();

Od niej wszystko sie zaczyna.

void wypisz_opcje(string napis,int n);

Otwiera i wypisuje zawartość pliku tekstowego jako menu do wybrania

void wypisz_w_kolorze(string napis,int n);

Wypisuje stringa w n kolorze

void menu(string napis,int n, int c);

Współpracuje z funkcja wypisz_opcje. W celu dodania interaktywnosci mozliwosc wybierania strzalkami w gore, dol oraz enterem. n - wybrane miejsce na którym jest kursor. c - czy czyścić ekran przy wywołaniu ( 0 = tak )

void menu_sortowania(string napis,int n,int c);

Wyświetla menu, po jakim elemencie struktury sortować (pesel/imie/nazwisko...). n - wybrane miejsce, na którym jest kursor. c - czy czyścić ekran przy wywołaniu ( 0 = tak ).

void menu_s_sortowania(string napis,int n,int wybor);

Wyświetla menu z rodzajem sortowania (rosnąco/malejąco/wróć).

n - wybrane miejsce, na którym jest kursor.

wybor - wartosc n z funkcji menu_sortowania (po jakim elemencie sortować).

void wybierz_sort(string napis,int r_m,int ktore_sortowac);

Uruchamia funkcje bubble_sort(napis,ktore_sortowac,r_m) lub menu_sortowania(napis,1) zależnie od wartości argumentu ktore_posortowac.

int il_linii(string napis);

Zwraca ilość linii w pliku tekstowym.

void wybierz_opcje(string napis,int n);

Wywołuje kolejne funkcje zależnie od wartości argumentu n

5. Instrukcja dla użytkownika

W celu wybrania jednej z możliwych opcji użytkownik nawiguję strzałkami w górę oraz w dół po czym wciska enter w celu wybrania dalszych możliwości.
Domyślny plik, na którym jest wykonywana praca to „data. dat” w celu jego zmiany należy przy uruchomieniu wybrać opcje nr 11.
W celu poprawnego wyświetlenia bazy danych (opcja nr 8) użytkownik powinien rozszerzyć okno swojej konsoli na tyle, aby w jednej linii można było zmieścić minimum 212 znaków. Jeżeli to nie jest możliwe proszę korzystać z opcji nr 2.
